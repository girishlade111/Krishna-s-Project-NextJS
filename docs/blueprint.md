# **App Name**: TUM Website Clone

## Core Features:

- Home Page Replication: Replicate the home page with pixel-perfect accuracy using HTML, CSS, and JS.
- Content Rendering: Add all content including text, images, and graphics exactly as available in the provided image.
- Typography Implementation: Implement the exact same typography including fonts, sizes, and styles shown in the image.
- Component Recreation: Recreate all components such as headers, footers, cards, and sections with exact fidelity.
- Size Adjustment: Make the website size exactly as shown in the original image.

## Style Guidelines:

- Primary color: Dark navy blue (#002E63) as the main background color, giving a professional and academic feel.
- Background color: Desaturated dark blue (#0A3A6E), providing depth while maintaining visual harmony.
- Accent color: A lighter blue (#4D7FB8), used for highlights and interactive elements, adding a touch of brightness and usability.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and readable design.
- Use a consistent set of clean, geometric icons to represent different sections and functions.
- Ensure a grid-based layout to maintain consistency and visual structure across the page.
- Apply subtle animations, such as fades and transitions, on interactive elements.
