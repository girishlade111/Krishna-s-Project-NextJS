# TUM Website Clone

This project is a clone of the Technical University of Munich (TUM) website, created within Firebase Studio. It's built using a modern tech stack including Next.js, React, TypeScript, and Tailwind CSS, with a focus on creating a pixel-perfect replica of the provided design.

## Key Features

*   **Pixel-Perfect Replica**: The layout, components, and styling have been meticulously crafted to match the design of the TUM website.
*   **Responsive Design**: The application is fully responsive, ensuring a seamless experience across desktops, tablets, and mobile devices.
*   **Interactive Elements**: Features dynamic components such as:
    *   A mobile-friendly navigation menu.
    *   Interactive video player modals.
*   **Component-Based Architecture**: The application is built with reusable React components using ShadCN UI for a clean and maintainable codebase.
*   **Modern Frontend Stack**:
    *   **Next.js**: For server-side rendering, routing, and an optimized developer experience.
    *   **React & TypeScript**: For building robust and type-safe user interfaces.
    *   **Tailwind CSS**: For utility-first styling and a consistent design system.

## Pages Created

*   `/`: The original landing page.
*   `/tum-clone`: A detailed page that clones the main landing page of the TUM website, featuring multiple sections for news, events, research, and community information.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You'll need to have Node.js and `npm` (or `yarn`) installed on your machine.

### Running the Development Server

1.  Install the project dependencies:
    ```sh
    npm install
    ```
2.  Start the Next.js development server:
    ```sh
    npm run dev
    ```
3.  Open [http://localhost:9002](http://localhost:9002) with your browser to see the result. You can view the main clone at [http://localhost:9002/tum-clone](http://localhost:9002/tum-clone).
